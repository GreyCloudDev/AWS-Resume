MovieFlixter is a mesh between IMBD and a Netflix clone that uses an API to sort through movies based on the category. In addition to giving you the Top10 movies titles, you can also receive upcoming. This API is created with Python, CSS, HTML and JavaScript.

The TODO Application is a CRUD application that allows you to create an account and curate a personalized todo list. You are able to create, review, update, and delete todo tasks. The list is stored in SQLite database that you can comeback to at a later date. This application is written using CSS, HTML, Python, Crispy-Forms, JQuery and SQLlite. You can view the demo here.
